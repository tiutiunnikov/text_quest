Написать текстовый квест

Структура задания:
1)отдельный класс UserInteface, который будет выводить тектовый консольный интерфейс

позабобиться о том, чтобы были проверки правильности ввода команд, и в случае если команда/вариант ответа не совпадает с ожидаемыми - переспросить еще реализовать


HelloGoodbye класс у вас должен вызываться в вашем основном файле (Game.ts/Main.ts , название не важно, но это файл который должен выполняться) для того, чтобы вы могли делать вывод приветствия и прощания


2) отдельный класс на сцену Scene - это класс, который должен будет отслеживать каш прогресс и соответсвенно заканчивать вашу игру


в классе должна быть своя переменная (объект этого класса), значение которой будет определять закочена игра или нет

пример:
export class Scene {
    progress:boolean = true;
}

или 
export class Scene {
    progress:boolean;

    constructor() {
        progress = true;
    }
}

проще всего progress переменную класса завязать с циклом while 


3) отдельный класс Levels ответственный за прохождение уровней, который собственно должен реализовать интерфейс уровней, т.е. у вас должен быть интефейс с объектами:
levelOne(): boolean;
levelTwo():boolean;
levelThree():boolean;



Структура файлов должна быть следующая(struct):
директория/папка gameLogics с файлами (interfaces.ts, scene.ts, levels.ts, userinterface.ts)
game.ts/main.ts 

Написать еще скрипнт в package.json который будет все компилить в отдельную директрию
соответственно, директория с исходниками у вас будет называться src, а директория с js файлами - bin

Т.е. в корне проекта должно быть так:
/src 
/bin (после компиляции)
/package.json
/tsconfig.json
